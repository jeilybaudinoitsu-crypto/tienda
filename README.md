# 🌸 Chroma DevPet

¡Bienvenido a **Chroma DevPet**! Una experiencia educativa interactiva diseñada para aprender a dominar las **Chrome DevTools** de una forma divertida, visual y moderna.

Chroma es una mascota virtual pastel que vive dentro de tu navegador y te guía paso a paso a través de los secretos de la inspección web y la consola de JavaScript.

---

## 🚀 ¿Qué es Chroma DevPet?

Esta aplicación combina **aprendizaje guiado** con **interactividad gamificada**. La idea es eliminar el miedo a la "consola técnica" y convertirla en una herramienta amiga.

### Características principales:
*   **Guía de Aprendizaje:** Tutorial paso a paso sobre cómo abrir la consola y entender las pestañas fundamentales: *Elements, Console, Network*.
*   **Mascota Virtual (Chroma):** Una guía interactiva en formato SVG con estados de ánimo, niveles de "amor" y animaciones personalizadas.
*   **Patio de Pruebas (Playground):** Interactúa con la consola real del navegador usando botones de alimentación que disparan comandos como `console.log`, `console.table`, `console.warn` y `console.error`.
*   **IA Chat Integrado:** Conectado a la API de **Gemini**, permite a los usuarios preguntar sobre errores, pedir trucos de CSS/JS o solicitar retos de programación.
*   **Diseño Estético (Pastel UI):** Una interfaz amigable, construida con Tailwind CSS, diseñada para reducir la carga cognitiva y hacer el aprendizaje más ameno.

---

## 🛠️ Tecnologías utilizadas

*   **HTML5 / CSS3:** Estructura y estilos personalizados con animaciones `keyframes`.
*   **Tailwind CSS:** Para un diseño responsivo, moderno y con paleta de colores pastel.
*   **JavaScript (Vanilla):** Lógica de control de la mascota, gestión de la consola y llamadas asíncronas.
*   **Lucide Icons:** Iconografía limpia y moderna.
*   **Google Gemini API:** El motor de inteligencia artificial que potencia el chat de Chroma.

## 📖 Cómo empezar

1.  **Clona este repositorio:**
```bash
    git clone [https://github.com/tu-usuario/chroma-devpet.git](https://github.com/tu-usuario/chroma-devpet.git)
    ```
2.  **Configura tu API Key:**
    Abre el archivo `index.html` y busca la variable `const apiKey = "";`. Pega tu clave de API de Google Gemini para habilitar el chat inteligente.
3.  **Abre el proyecto:**
    Simplemente abre el archivo `index.html` en tu navegador favorito (Chrome recomendado) y presiona `F12` para ver la consola en acción.

---

## 🎮 Cómo jugar con Chroma

1.  **Aliméntala:** Haz clic en los botones de comida (🍪, 🍦, 🍋, 🌶️). Cada uno dispara un método diferente de la consola (`console.log`, `console.info`, etc.).
2.  **Observa la consola:** Mira cómo tus acciones cambian el estado de la aplicación y disparan logs en tiempo real.
3.  **Pregúntale a la IA:** Ve a la pestaña "IA Chat" y pídele retos o ayuda con un código que no entiendas.
